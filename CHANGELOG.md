# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),

## [0.1.0] - 2024-XX-XX

### Adicionado
- Suporte para cookiecutter no download do framework T2C
- Comando CLI `t2c` (além de `t2c-speckit`)
- Integração com repositório do framework T2C: https://github.com/T2C-Consultoria/prj_botcity_framework_template.git
- Preparado para uso via uvx: https://github.com/PROJETOS-IA-T2C/T2C_SpecKit.git
- CLI principal com comando `init`
- Interface interativa para seleção de AI Assistant
- Extração de DDP de arquivos PPTX
- Geração completa do framework T2C
- Templates de especificação (spec, plan, selectors, business-rules, tasks)
- Comandos Cursor para extração, geração de tasks, implementação e validação
- Scripts PowerShell e Bash de automação
- Constitution do framework T2C
- Validação de especificações
- Geração de código customizado baseado em specs

### Documentação
